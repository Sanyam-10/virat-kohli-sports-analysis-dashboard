🏏 Virat Kohli Sports Analysis Dashboard

An interactive Power BI dashboard analyzing Virat Kohli’s cricket career across ODIs, Tests, and T20s. The dashboard provides insights into his performance, trends, and key statistics over the years.

📊 Features

Total runs scored across formats

Batting average and strike rate analysis

Centuries and half-centuries trends

High scores by year and match format

Performance against top opponents

Venue-wise performance insights

🛠 Technologies Used

Power BI Desktop – For building interactive dashboards

Microsoft Excel – Data cleaning and preprocessing

DAX – Calculated columns and measures

Power Query – Data transformation
